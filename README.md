## 背景
学校里学完csapp之后就对操作系统很感兴趣,一直想自己实现一个操作系统,但是感觉好难好难,直到看到了清华大学的ucore实验,也是自己实现一个操作系统,但是自己实现的部分太少了
绝大部分都是已经完成了的,做起来没什么成就感,后来又听说同学做了mit6.828,研究了一下发现不错,就着手来做了


## 关于ucore_lab
